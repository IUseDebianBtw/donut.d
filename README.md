Really putting the D in Donut :trollface:

## Usage

```$ git clone https://github.com/IUseDebianBtw/donut.d.git```

```$ cd donut.d```

```$ gdc donut.d -o donut```

or

```$ dmd donut.d```

```$ ./donut```
